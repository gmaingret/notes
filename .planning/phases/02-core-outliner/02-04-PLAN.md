Props: `{ bullet: FlatBullet; bulletMap: BulletMap; onFocus?: () => void }`

Internal state: `localContent` (string) mirroring `bullet.content`, updated immediately on `onInput`. `saveTimer` ref for debounce.

On `onInput`: update `localContent`, clear and restart 1000ms timer that calls `patchBullet({ id: bullet.id, content: localContent })` and then `undoCheckpoint({ id: bullet.id, content: localContent })`.

Cursor preservation: do NOT update `localContent` from props while the bullet is focused (check `document.activeElement === divRef.current`). Only sync from props when not focused.

`handleKeyDown(e)`:
- `Enter` (not Shift): `e.preventDefault()`. Check: empty content AND parentId not null → `outdentBullet(bullet.id)`; empty content AND parentId null → `createBullet({ documentId, parentId: null, afterId: bullet.id, content: '' })`; else if children exist → `createBullet({ documentId, parentId: bullet.id, afterId: null, content: after })` (first child); else → `createBullet({ documentId, parentId: bullet.parentId, afterId: bullet.id, content: after })` (sibling). `splitAtCursor` extracts before/after. Update current bullet content to `before`. After create mutation resolves, `document.getElementById(newBulletId)?.focus()`.
- `Tab` (not Shift): `e.preventDefault()`. If bullet is first sibling (no previous sibling in bulletMap), silent no-op. Else `indentBullet(bullet.id)`.
- `Shift+Tab`: `e.preventDefault()`. If parentId null, silent no-op. Else `outdentBullet(bullet.id)`.
- `Backspace` at cursor start: `e.preventDefault()`. If bullet has children (check bulletMap), add class 'shake' for 400ms. Else call merge-up logic: `patchBullet({ id: prevBullet.id, content: prevContent + localContent })` then `softDeleteBullet(bullet.id)`; focus prev bullet and set cursor to join point.
- `Delete` at cursor end: `e.preventDefault()`. Mirror Backspace-merge: merge next bullet's content into current, delete next bullet.
- `Ctrl/Cmd+ArrowUp`: `e.preventDefault()`. `moveBullet({ id: bullet.id, newParentId: bullet.parentId, afterId: prevSiblingOfPrevSibling })`.
- `Ctrl/Cmd+ArrowDown`: `e.preventDefault()`. `moveBullet({ id: bullet.id, newParentId: bullet.parentId, afterId: nextNextSibling })`.
- `Ctrl/Cmd+B`: `e.preventDefault()`. Wrap selection with `**`. Use `window.getSelection()` + `Range` to read selected text, replace with `**${selected}**` in textContent.
- `Ctrl/Cmd+I`: Same but `*${selected}*`.

Helper functions (export for tests): `splitAtCursor(el)`, `isCursorAtStart(el)`, `isCursorAtEnd(el)` — use exact implementations from RESEARCH.md Pattern 7.

CSS for shake: inline `@keyframes shake { 0%,100%{transform:translateX(0)} 25%{transform:translateX(-4px)} 75%{transform:translateX(4px)} }` as a style tag or Tailwind arbitrary. Apply class on state trigger.
