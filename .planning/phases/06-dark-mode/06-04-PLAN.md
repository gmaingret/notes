    New head content (exact order matters):
    ```html
    <head>
      <meta charset="UTF-8" />
      <meta name="color-scheme" content="light dark" />
      <script>
        if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
          document.documentElement.classList.add('dark');
        }
      </script>
      <link rel="icon" type="image/svg+xml" href="/vite.svg" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <title>Notes</title>
    </head>
    ```

    Changes:
    1. Add `<meta name="color-scheme" content="light dark" />` as the second tag (after charset, before anything else).
    2. Add the inline `<script>` block immediately after the color-scheme meta, before the icon link.
    3. Change `<title>client</title>` to `<title>Notes</title>` (correct app name while here).

    The script MUST be synchronous (no defer, no type="module"). It MUST appear before any `<link rel="stylesheet">` tags (there are none currently since Vite injects CSS at build time — but the position before other links is still correct for future changes).

    After editing, run the automated test to confirm DRKM-03 and DRKM-04 tests now pass.
