       - `drawerContent = { ModalDrawerSheet { DocumentDrawerContent(...) } }`

       - `onDelete`: set `deleteConfirmation = doc` and close drawer first (so dialog renders over content, not drawer). Find the Document object from uiState.documents by ID.
       - `onRename`: call `viewModel.startRename(docId)` (doc stays in drawer, TextField appears)
       - `onCreateDocument`: call `viewModel.createDocument()` (new doc appears at end with inline edit active)

         - `actions`: overflow MoreVert menu with "Log out" (same as current, calls `viewModel.logout(onComplete = onLogout)`)

    6. **Content area** (inside Scaffold):
       - If `uiState is Loading`: centered `CircularProgressIndicator()`
       - If `uiState is Error`: centered `Text(uiState.message)` with retry button
       - If `uiState is Empty`: centered `Text("No documents")` with "Create document" button (convenience duplicate of drawer's empty state)
       - If `uiState is Success` and `openDocumentId != null`: Column with document title as `Text` using `MaterialTheme.typography.headlineMedium`, and below it `Text("Content area — Phase 11 will add the bullet tree editor here", style = MaterialTheme.typography.bodyMedium, color = MaterialTheme.colorScheme.onSurfaceVariant)`
