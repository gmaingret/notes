    **BulletTreeScreen.kt updates — drag-reorder with reparenting integration:**

    Replace plain LazyColumn with Reorderable-wrapped version (same pattern as DocumentDrawerContent from Phase 10). Track horizontal drag offset for reparenting using a `Modifier.pointerInput` on the drag handle that accumulates horizontal displacement.

    ```kotlin
    val lazyListState = rememberLazyListState()
    // Track horizontal offset during drag for reparenting
    var dragHorizontalOffset by remember { mutableFloatStateOf(0f) }
    var draggedBulletId by remember { mutableStateOf<String?>(null) }

    val reorderableState = rememberReorderableLazyListState(lazyListState) { from, to ->
        viewModel.moveBulletLocally(from.index, to.index)
    }

    LazyColumn(
        state = lazyListState,
        modifier = Modifier.fillMaxSize().imePadding()
    ) {
        items(flatList, key = { it.bullet.id }) { flatBullet ->
            ReorderableItem(reorderableState, key = flatBullet.bullet.id) { isDragging ->
                // Show drop indicator line offset by target depth when dragging
                val indentPerLevel = 24.dp
                val targetDepthDelta = if (isDragging) {
                    (dragHorizontalOffset / with(density) { indentPerLevel.toPx() }).roundToInt()
                        .coerceIn(-flatBullet.depth, 1) // can't outdent past root, can indent max 1 level
                } else 0

                BulletRow(
                    flatBullet = flatBullet,
                    isDragging = isDragging,
                    modifier = Modifier
                        .animateItem()
                        .longPressDraggableHandle(
                            onDragStarted = {
                                view.performHapticFeedback(HapticFeedbackConstants.LONG_PRESS)
                                draggedBulletId = flatBullet.bullet.id
                                dragHorizontalOffset = 0f
                            },
                            onDragStopped = {
                                // Compute reparenting from horizontal offset using projection algorithm
                                val bullet = flatBullet.bullet
                                val currentIndex = flatList.indexOfFirst { it.bullet.id == bullet.id }
                                val targetDepth = (flatBullet.depth + targetDepthDelta).coerceAtLeast(0)

                                // Find newParentId by walking up from drop position to find bullet at targetDepth-1
                                var newParentId: String? = null
                                if (targetDepth > 0 && currentIndex > 0) {
                                    for (i in (currentIndex - 1) downTo 0) {
                                        if (flatList[i].depth == targetDepth - 1) {
                                            newParentId = flatList[i].bullet.id
                                            break
                                        }
                                    }
                                }

                                // Find afterId (previous sibling at same depth under same parent)
                                var afterId: String? = null
                                if (currentIndex > 0) {
                                    for (i in (currentIndex - 1) downTo 0) {
                                        val candidate = flatList[i]
                                        if (candidate.bullet.parentId == newParentId && candidate.depth == targetDepth) {
                                            afterId = candidate.bullet.id
                                            break
                                        }
                                        if (candidate.depth < targetDepth) break
                                    }
                                }

                                viewModel.commitBulletMove(
                                    bulletId = bullet.id,
                                    newParentId = newParentId,
                                    afterId = afterId
                                )
                                draggedBulletId = null
                                dragHorizontalOffset = 0f
                            }
                        )
                        .pointerInput(Unit) {
                            // Track horizontal displacement during drag for reparenting depth
                            detectDragGestures { _, dragAmount ->
                                if (draggedBulletId == flatBullet.bullet.id) {
                                    dragHorizontalOffset += dragAmount.x
                                }
                            }
                        },
                    // ... other params
                )
            }
        }
    }
    ```

    **Drop indicator:** When dragging, show a horizontal line at the drop position. The line's left offset = `targetDepth * indentPerLevel` to visually communicate the reparenting depth. Implement as a thin `Divider` or `Box(height=2.dp, background=primary)` rendered conditionally when `isDragging` is true on adjacent items.

    **Cycle prevention:** Before committing the move, check that `newParentId` is not a descendant of the dragged bullet. Build a descendant set from the flat list by collecting all bullets between the dragged bullet and the next bullet at same-or-lower depth. If `newParentId` is in the descendant set, skip the move and show a Snackbar "Cannot move bullet under its own child". See RESEARCH.md Pitfall 7.
