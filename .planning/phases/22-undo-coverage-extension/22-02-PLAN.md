5. **Record a single undo event with batch ops** — inside the transaction, after the soft delete:
```typescript
const forwardOps = completedBullets.map(b => ({ type: 'delete_bullet' as const, id: b.id }));
const inverseOps = completedBullets.map(b => ({ type: 'restore_bullet' as const, id: b.id }));

await recordUndoEvent(
  tx,
  user.id,
  'bulk_delete_completed',
  { type: 'batch', ops: forwardOps },
  { type: 'batch', ops: inverseOps }
);
```

This records ONE undo event. Undoing it runs `restore_bullet` for each deleted bullet (sets `deletedAt = null`). Redoing runs `delete_bullet` for each (sets `deletedAt = new Date()`).
